# Best by Use Case

This page is intentionally opinionated.

It is not a ranking of the "best AI agents" overall.
It is a shortlist of good starting points depending on what you are actually trying to do.

## Best starting points for coding

### If you want the strongest repo-scale coding help
- **Claude Code**
- **Cursor**
- **Codex CLI**
- **Gemini CLI**

### If you want open-source coding tools
- **Aider**
- **OpenHands**
- **Cline**
- **Continue**
- **OpenCode**
- **Crush CLI**
- **ForgeCode**
- **Plandex**

### If you care about git or PR-friendly workflows
- **Aider**
- **Sweep**
- **CodeRabbit**
- **Amp**

## How to read this page

Use it as a starting map, not final truth.

The right tool depends on:
- whether you want local or cloud
- whether you want open source or managed software
- whether you want a product, framework, or infrastructure layer
- whether you want speed, control, or ease of use
